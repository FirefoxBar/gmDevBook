简介

和其它 API 不同，这是一个变量而非函数。

该变量提供当前执行的用户脚本的元数据。

语法

GM_info

返回

该对象含有下述属性：

script
一个包含当前脚本的 元数据。
name
文本
namespace
可能为空文本。
description
可能为空文本。
run-at
文本
version
可能为空文本
includes
可能为空数组。
excludes
可能为空数组。
matches
@match，可能为空数组 (0.9.16 下有问题)
resources
包含所有资源和其对应的地址的对象 (自 GM 1.2)。
unwrap
布尔型
scriptMetaStr
文本，元数据文本原文 (不含分隔符)
scriptWillUpdate
如果修改为 true，GM 将尝试自动更新脚本。
version
GM 版本。如 0.9.16
