描述

该函数用于获取定义的 @resource 的元属性值。

语法

声明的资源名称。

返回

String

异常

当资源不存在的时候将抛出异常。

例子

// ==UserScript==
// 首先定义一个资源
// @resource prototype http://www.example.com/prototype.js
// ==/UserScript==

// 然后从这里获取资源地址
var prototypeSource = GM_getResourceText("prototype");

笔记

  • 添加 !important 到 CSS 末尾可强行覆盖其属性值。
  • 如有需要，可使用「多行文本」可能方便排版。
