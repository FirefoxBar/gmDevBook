描述

该函数用于获取脚本之前使用 GM_setValue 赋值储存的数据，可以为 String、Boolean 等类型。

语法

欲获取数据的名称。 如果上述数据不存在时返回的值。

返回

该函数将返回获取到的值，如果不存在则返回 default 传入的值。

例子

抓取一个叫做 foo 的储存值

// 输出储存为 foo 的值或 undefined

GM_log(GM_getValue("foo"));

抓取一个叫做 'timezoneOffset' 的值并提供默认值:

// 如果该值不存在 GM_getValue() 则会返回 +8 (整数)

GM_log(GM_getValue("timezoneOffset", +8));
