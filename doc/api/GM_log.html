描述

一个简单的日志函数，用于输出数据至控制台。按下键盘 F12 查看。

输出数据将输出至「错误控制台」，包括所属脚本命名空间以及传入的文本。

因此，下述脚本：

// ==UserScript==
// @name          GM_log Example
// @namespace     http://www.example.com/
// ==/UserScript==

GM_log("This is an example of GM_log");

将产生这样的输出：

http://www.example.com/GM_log Example: This is an example of GM_log

语法

用于输出的文本内容，也可以为带有 .toString () 方法的变量。

返回

undefined

例子

GM_log("Hello, World!");
GM_log("Warning, " + someInputField.value + "!");
